## 1 TA-Lib 模块

1.1 TA-Lib 安装


TA-Lib 模块有很多版本，因此， TA-Lib 安装过程中可能会遇到各式各
样的问题。一般情况下首次通过 pip 命令进行安装。在 Window 环境
下，通常可以在 DOS 命令行终端执行 pip 命令行完成安装。即在
Win10 命令窗口输入代码 pip3 install TA-Lib 即可。


注意 :

   - 安装时， TA-Lib 模块名称注意区分大小写，如果将“ TA-Lib" 写成
“ talib” 则会报错 ( 注意 :TA-Lib 模块安装好后，引入 TA-Lib 模块则
应输入“ import talib” ，而非输入“ import TA-Lib”) 。

   - 如果出现报错，则考虑通过下载 whl 文件后再通过 pip 安装方式完
成安装。
