TA-Lib模块有很多版本，因此，TA-Lib安装过程中可能会遇到各式各样的问题。一般情况下首次通过pip命令进行安装。在Window环境下，通常可以在DOS命令行终端执行pip命令行完成安装。即在Win10命令窗口输入代码pip3 install TA-Lib即可。

如果出现报错，则考虑通过下载whl文件后再通过pip安装方式完成安装。

1.计算 MACD 指标使用 talib.MACD() 函数计算 MACD 指标，函数的输入是股票的收盘价（df_stockload['close'].values），并设置以下参数：
fastperiod=12：快速线的周期为12
slowperiod=26：慢速线的周期为26
signalperiod=9：信号线的周期为9
该函数返回三个值：
macd_dif：差离值（DIF线），表示短期和长期均线的差距。
macd_dea：信号线（DEA线），是DIF线的平滑平均。
macd_bar：柱状图值，表示DIF与DEA的差。

2.绘制 DIF 和 DEA 线
使用 ax.plot() 函数分别绘制DIF线（红色）和DEA线（蓝色）。
这两条线用于展示MACD指标中的短期与长期趋势。

3.绘制 MACD 柱状图
对于 macd_bar，如果值大于0（表示市场处于上涨状态），则将柱子绘制为红色；
如果值小于0（表示市场处于下跌状态），则将柱子绘制为绿色。
使用 np.where() 来生成两组柱状图的值：bar_red 和 bar_green，分别对应红色和绿色的柱状图。
ax.bar() 用于绘制这两组柱状图，并且设置柱子的宽度为0.6，确保它们与之前的成交量柱状图宽度一致。

4.2 KDJ绘图(补充内容)

使用 talib.STOCH() 函数计算 Stochastic 指标（K, D, J），其中：
输入数据为：股票的最高价（df_stockload.high.values）、最低价（df_stockload.low.values）、收盘价（df_stockload.close.values）。
设置以下参数：
fastk_period=9：K线的计算周期为9。
Slowk_period=3：K线的平滑周期为3。
slowk_matype=0：K线使用简单移动平均（SMA）。
slowd_period=3：D线的平滑周期为3。
slowd_matype=0：D线使用简单移动平均（SMA）。
df_stockload['K']：K线值，表示股票价格的相对位置，通常用于短期的超买超卖判断。
df_stockload['D']：D线值，为K线的平滑值，通常用于长期趋势判断。
df_stockload['J‘]：使用公式 J = 3 * K - 2 * D 计算 J线，该线是一种在 Stochastic 指标上进一步增强的信号，常用于判断超买超卖和趋势反转的潜力。
使用 ax.plot() 函数分别绘制 K 线（蓝色），D 线（绿色虚线），和 J 线（红色实线）。这些线能够展示股票的超买超卖情况，以及价格走势的潜在反转信号。

1.计算 RSI 指标
使用 talib.RSI() 函数计算相对强弱指标（RSI）。该函数的输入是股票的收盘价（df_stockload['close'].values），并设置计算周期为14（timeperiod=14），即标准的14日RSI。RSI指标用于衡量股票在一段时间内的价格变动强度，通常用于识别超买和超卖状态。

2.绘制 RSI 曲线
使用 ax.plot() 函数绘制 RSI 指标的线，设置为紫色，并给它添加标签 "RSI(14)"。lw=1.0 设置线宽为1，以便清晰显示。

3.添加超买超卖线
使用 ax.axhline() 函数在图表上绘制水平线，标示超买和超卖区间：ax.axhline(70, linestyle='--', color='red', alpha=0.5)：红色虚线表示超买区，RSI超过70时，表示市场可能过热。ax.axhline(30, linestyle='--', color='green', alpha=0.5)：绿色虚线表示超卖区，RSI低于30时，表示市场可能被过度抛售。

4.填充超买超卖区域
使用 ax.fill_between() 函数填充超买和超卖区域，以便更直观地展示市场过热或过冷的状态：当 RSI 大于或等于 70 时，填充红色区域（超买区域），使用 color='red' 和 alpha=0.3 来控制透明度。当 RSI 小于或等于 30 时，填充绿色区域（超卖区域），使用 color='green' 和 alpha=0.3 来控制透明度。
